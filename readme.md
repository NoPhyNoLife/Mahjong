# readme.md
player_id 是当前玩家：
    本家为0，
    上家为3，
    下家为1，
    对家为2。

牌组及其编码：（每张牌面均有四张同样的牌）
    饼系列： B1 - B9, 对应 [0,1] - [0,9] 
    万系列： W1 - W9, 对应 [1,1] - [1,9]
    条系列： T1 - T9, 对应 [2,1] - [2,9]
    “东南西北中发白”：
        东 E, 对应 [3,0]
        南 S, 对应 [4,0]
        西 W, 对应 [5,0]
        北 N, 对应 [6,0]
        中 M, 对应 [7,0]
        发 R, 对应 [8,0]
        白 B, 对应 [9,0]
